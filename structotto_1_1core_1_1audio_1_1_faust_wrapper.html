Detailed Description

template<int Cin, int Cout>
struct otto::core::audio::FaustWrapper< Cin, Cout >

A Wrapper for faust scripts All interactions with faust should go through this wrapper

Template Parameters
CinThe number of input audio channels. Make sure this corresponds with the actually called faust script
CoutThe number of output audio channels Make sure this corresponds with the actually called faust script
