otto::util::double_buffered< T, AfterSwap > Struct Template Reference

Detailed Description

template<typename T, typename AfterSwap = clear_outer>
struct otto::util::double_buffered< T, AfterSwap >

Inner and outer buffers swapped atomically.

The inner buffer can always be accessed lock-free, the outer is locked on swap.

Template Parameters
Tneeds .clear(), which is called on swap
