otto::board::ui::RpiFBCP Struct Reference

Detailed Description

This class is used to copy from the HDMI framebuffer on the pi, to the framebuffer which is used by fbtft.

This is useful when using an SPI display.

At some point in the future, we will probably want to skip some of these steps, possibly by including the spi driver directly and writing the main framebuffer to that on each frame.
