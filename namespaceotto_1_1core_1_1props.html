enum otto::core::props::HookOrder
strong

Order of hook handlers.

This is the second template argument to hook implementations, and is used to sort handlers.

Enumerator
Before 

Should only read hook values.

Early 

Change values early.

Middle 

Change values in the middle. Default.

Late 

Change values late.

After 

Should only read hook values.
