Reflection

Detailed Description

The OTTO reflection library is based on MetaStuff by Elias daler.

You register classes for reflection using the macros DECL_REFLECTION or DECL_REFLECTION_OUTSIDE, or by manually declaring the member reflect_members or nonmember register_members, along with reflect_name and register_name

Reflection is mainly used for serialization, using otto::util::serialize and otto::util::deserialize, but can be used for other purposes as well. The main function is reflect::for_all_members, which can be used to iterate over members of a struct.
