Detailed Description

template<typename... Iterators>
struct otto::util::iterator::zipped_iterator< Iterators >

Zipped iterator.

Iterates over multiple iterators at the same time

Its very useful with structured bindings and for loops. Using the helper function zip(), you can iterate over multiple ranges like this:

for (auto&& [r1, r2] : util::zip(range1, range2)) {
...
}
