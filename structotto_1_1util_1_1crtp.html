Detailed Description

template<typename Derived, typename CrtpT>
struct otto::util::crtp< Derived, CrtpT >

A helper for CRTP.

Exposes a {derived()} function to the CRTP class only, not to types deriving from the CRTP type.

Usage:

template<typename Derived, typename SecondT>
struct SomeBase : crtp<Derived, SomeBase<Derived, SecondT>> {}
Template Parameters
Derivedthe "real" derived type
CrtpTthe CRTP type
