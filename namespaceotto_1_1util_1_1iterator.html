otto::util::iterator Namespace Reference

Detailed Description

This namespace contains custom iterator types and general helper classes for working with containers

Attention
otto::util::iterator is an inline namespace, meaning all members can and should be accessed directly from the otto::util namespace i.e. as util::float_step(...), not util::iterator::float_step(...). It is only a separate namespace for clarification of documentation and name resolution.
