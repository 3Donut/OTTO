void otto::services::GLFWUIManager::main_ui_loop ( )
overridevirtual

The main ui loop.

This sets up all the device specific graphics, and calls draw_frame 60 times pr second, until Application::running() is false, or the graphics are exitted by the user. It is also responsible for listening to keyevents, and calling keypress and keyrelease as apropriate.

On some platforms (OSX), all OpenGL calls must be made from the main thread, therefore this function is called from main().

Implements otto::services::UIManager.
