otto::util::result< ok_t, err_t > Struct Template Reference

Detailed Description

template<typename ok_t, typename err_t>
struct otto::util::result< ok_t, err_t >

A sum type, holding either an Ok value, or an Err.

Modelled closely after Rust's Result enum.

Example

```cpp result<std::string, int> open_file(File& f) { f.open(); if (f.error()) { return f.errorCode(); } return f.path(); } ```
