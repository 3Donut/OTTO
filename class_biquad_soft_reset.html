Detailed Description

template<class Tv = gam::real, class Tp = gam::real, class Td = gam::DomainObserver>
class BiquadSoftReset< Tv, Tp, Td >

2-pole/2-zero IIR filter with soft reset

The biquadratic filter contains 2 zeros and 2 poles in its transfer function. The zeros provide a better response near the DC and Nyquist frequencies than an all-pole filter would. Second-order IIRs have a 12 dB/octave cutoff slope and are normally cascaded (run in series) to obtain a sharper response. This particular implementation utilizes the RBJ Sallen-Key/Butterworth design from:

http://www.musicdsp.org/files/Audio-EQ-Cookbook.txt
